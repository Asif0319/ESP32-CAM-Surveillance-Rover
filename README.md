# ESP32-CAM-Surveillance-Rover
A tiny, battery-operated robot automobile with a live-streaming camera is called the ESP32-CAM Surveillance Rover.  It may be operated from any web browser and monitors the surroundings and detects impediments using sensors including light, gas/smoke, and ultrasonic.
